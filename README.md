## Installation

[![NPM](https://nodei.co/npm/stvl.png?downloads=true&downloadRank=true&stars=true)](https://www.npmjs.com/package/stvl)

```
npm install stvl
```

**Via [Yarn](https://yarnpkg.com):**

```
yarn add stvl
```
